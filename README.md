\# HTML Quick-Styler



AI-Powered HTML Page Generator using IBM Granite 3.3 2B Instruct and Gradio.



\## Project Description



HTML Quick-Styler is an intelligent web page generation platform that automatically creates modern and responsive HTML websites based on user input.



The system uses the IBM Granite 3.3 2B Instruct model to generate structured web page content and combines it with a dynamic CSS styling engine to produce professional website layouts.



Users simply enter a page title, description, and select a color theme. The application then generates a complete HTML webpage with multiple sections such as hero section, features, about, portfolio, testimonials, and contact form.



The interface is built using Gradio, which provides an interactive web UI for generating and previewing websites instantly.



---



\## Features



\- AI-powered HTML page generation

\- Automatic website section creation

\- Modern CSS styling engine

\- 8 built-in color palettes

\- Responsive mobile-friendly design

\- Interactive web interface using Gradio

\- Live website preview
